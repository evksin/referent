PROJECT.md - описание проекта

## Установка

Установите зависимости:

```powershell
pnpm install
```

## Запуск

Запустите сервер разработки:

```powershell
pnpm dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## Сборка

Для создания production сборки:

```powershell
pnpm build
pnpm start
```
